##3. **Zero** standard deviation and **NA** standard deviation

If a carrier has only **one observation**, the standard deviation formula divides by ( n - 1). Since (n = 1) we get **division by zero**, which results in `NA`.

na_summaries <- flights |> filter(month ==1 ,carrier=='OO') |> summarise( Number_of_flights = n(),
    mean_dist = mean(distance, na.rm = TRUE),
    sd_dist = sd(distance, na.rm = TRUE))
